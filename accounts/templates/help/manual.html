Record Types

There are 3 types of records for any record submitted:

Proposal

-Records that are to be proposed to the advisers. This is a pre-requisite for "Thesis/Research" record type
-Can be created by any non-guest users

Thesis/Research

-Must have submitted a proposal already.
-This is automatically grouped together with the "Proposal" record so that you can easily navigate from "Proposal" to "Thesis/Research" or vise versa
-Only the owners of the record with "Proposal" record type can add a new record with "Thesis/Research" record type
-Users can only add the "Thesis/Research" record type from the opened "Proposal" record

Project

-Other records
-Can be created by any non-guest users

Record submission flow

Student > Adviser > KTTO / TBI > RDCO

  1. Student can submit 1 or more records to 1 specific adviser only
  2. Adviser can then review the submitted record and submit it to the KTTO
  3. KTTO / TBI users shares the same records submitted to them. KTTO/TBI users automatically receives the submitted record from the advisers, reviewed and are then submitted to KTTO/TBI users.

Managing Records

Adding Records

Note: guest users are not permitted to add a record
There are 2 ways to add a record:

  • Add button on the records tab from the homepage
  • By migrating records from excel file

Editting Records

  • For Student / Adviser
    • -Only the owner of the record can edit the record but only if it hasn't been approved yet by an adviser
  • ITSO / KTTO / RDCO
    • -Can edit any records anytime from the "Dashboard > Manage Records" or records that it owns

Deleting Records

  • if the record is approved, the record can no longer be deleted
  • if the record is declined or is not yet approved, the record can be deleted
